项目简介

MovieVizCrawler 是一个基于 Flask 的豆瓣电影 TOP250 数据可视化平台。 它抓取豆瓣电影 TOP250 榜单数据，提供数据浏览、多维度统计分析和 RESTful API 服务。

技术栈
  • 后端： Flask + SQLAlchemy + SQLite
  • 前端： Bootstrap + jQuery + ECharts
  • 数据源： 豆瓣电影 TOP250
功能特性
  • 电影列表浏览，支持关键词搜索、评分筛选、类型筛选
  • 电影详情页，展示完整信息
  • 多维度数据分析（评分分布、年份趋势、类型占比）
  • 完整 RESTful JSON API
  • 响应式设计，支持移动端
API 接口

项目提供 RESTful API，接口地址以 /api/ 开头：

接口 说明
GET /api/movies电影列表（支持搜索筛选）
GET /api/movies/<id>电影详情
GET /api/stats/score评分分布
GET /api/stats/year年份分布
GET /api/stats/genre类型分布
GET /api/overview数据概览
数据来源

电影数据来源于 豆瓣电影 TOP250， 仅用于学习和展示目的。
